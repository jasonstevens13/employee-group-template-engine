# Employee-Group-Template-Engine
A node.js cli app to take in a work team and generate an html page dynamically, to display the team and their contact information.

## Description
  This node.js command line application allows the user to create an html file that contains a working teams contact information. The app takes in a manager's name and informations and then allows the user to buld the rest of the team of Engineers and/or Interns through cli prompts.  
  
## Installation
  The intention is to make this available via an npm, but for now one may simply grab the code from GitHub. The app utlizes the following  dependencies: inquirer; path; fs; and jest.
  
## Usage
  Any/All developers work on teams. A simple link to an html page containing all team members information allows for quick and easy communication. This node application allows the dev to create the html file in a templated fashion, through the command line. The skeleton of the html page, and the styling contained therein, are all done for the user.
